Objetivo:
	O objetivo do programa é para a organização de arquivos em uma pasta, separando em pastas posteriomente
criadas pelo próprio programa, chamadas de: Fotos, Vídeos, Documentos, Músicas e Scripts. Os arquivo serão
separados pelas suas respectivas extensões. Vários tipos suportados, tais eles são:
 |-------------------------------------------------------------------------------------|
 | Fotos |   Video |  Documentos  |  Músicas  |  Scripts |  Compactados |  Executáveis |
 |------ |---------|--------------|-----------|----------|--------------|--------------|
 | .jpg  |  .wmv   |   .doc       |   .mp3    |   .sh    |    .rar      |    .exe      |
 | .png  |  .mp4   |   .docx      |   .wav    |   .bat   |    .zip      |    .msi      |
 | .bmp  |  .mkv   |   .txt       |   .flac   |   .py    |    .7zip     |    .run      |
 | .jpeg |  .rmvb  |   .odt       |   .aac    |   .java  |    .tar.gz   |              |
 | .gif  |  .flv   |              |           |          |    .tar.gx   |              |
 |       |  .3gp   |              |           |          |              |              |
 |       |  .avi   |              |           |          |              |              |
 |       |  .mov   |              |           |          |              |              |
 |       |         |              |           |          |              |              |
 |-------------------------------------------------------------------------------------|
